# Choser

Choser is a minimal mobile app designed to help you make small decisions faster.
Instead of wasting mental energy choosing between many options, you can create a list and let the app randomly select one for you.

The idea is simple: delegate trivial decisions to randomness so you can focus on what actually matters.

## Features

* Create multiple lists
* Add and remove items easily
* Randomly pick a winner from a list
* Smooth animated draw
* Simple and distraction-free interface

## Example Use Cases

Choser can help you decide things like:

* What movie to watch
* What game to play
* Where to eat
* What task to start next
* Any situation with too many options

## Tech Stack

* **Flutter**
* **Dart**

The app is designed to be lightweight and easy to maintain.

## Motivation

Modern apps bombard users with options. Choosing between many possibilities often leads to decision fatigue.
Choser exists to offload those trivial choices into a simple randomized system.
